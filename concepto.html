    ¿QUÉ ES SEGURIDAD INFORMÁTICA?

    La seguridad informática pretende identificar las amenazas y reducir los riesgos al detectar las vulnerabilidades acabando o minimizando así el impacto o efecto nocivo que se pueda causar.

    La seguridad informática no es un producto, Es el proceso mediante el cual se protege la información almacenada en un sistema informático, c on el fin de prevenir y detectar la violación a un sistema evitando que personas maliciosas puedan obtener acceso a la información usándola para fines económicos o simplemente que puedan dañar la información por un acceso accidental o voluntario a los sistemas. Se enfoca en la protección de la infraestructura computacional y todo lo relacionado con esta especialmente, la información contenida o circulante. Para ello existen una serie de estándares, protocolos, métodos, reglas, herramientas y leyes concebidas para minimizar los posibles riesgos a la infraestructura o a la información. La seguridad informática comprende software (bases de datos, metadatos, archivos), hardware y todo lo que la organización valore y signifique un riesgo si esta información confidencial llega a manos de otras personas. Las cuatro áreas principales que cubre la seguridad informática son: -Confidencialidad -Integridad -Disponibilidad -Autenticación

    TIPOS DE SEGURIDAD INFORMÁTICA

    Existen tres tipos de seguridad informática en la que debemos enfocar nuestra atención con el fin de evitar pérdidas de información valiosa que puede afectar la credibilidad y reputación de cualquier empresa. Estas son:

    SEGURIDAD DE HARDWARE

    Este tipo de seguridad se enfoca en la protección de los equipos informáticos, estos son muy vulnerables los dispositivos de entrada y salida conectados a la red pueden producir daños graves, también las fuentes como el código que se ejecuta en el hardware.

    SEGURIDAD DE SOFTWARE

    Se usa para proteger el software de ataques maliciosos, para que funcione de manera correcta. Los agujeros de seguridad del software son muy comunes ya que las aplicaciones tienen salida a internet. El software debe ser actualizado constantemente para lograr protegerse de amenazas emergentes.

    SEGURIDAD DE RED

    Se refiere a cualquier actividad diseñada para proteger la red. Entre las amenazas más comunes se encuentran Virus, gusanos y caballos de Troya, Software espía y publicitario, ataques de día cero, también llamados ataques de hora cero, Ataques de hackers, Ataques de denegación de servicio, Intercepción o robo de datos, Robo de identidad. Se deben tener varios niveles de seguridad con eso si uno falla el otro sigue en pie defendiendo la red. Un sistema de seguridad de red tiene muchos componentes de seguridad que incluyen, Antivirus y antispyware, Cortafuegos, para bloquear el acceso no autorizado a su red, Sistemas de prevención de intrusiones (IPS), para identificar las amenazas de rápida propagación, como el día cero o cero horas ataques Redes privadas virtuales (VPN), para proporcionar acceso remoto seguro.
